# Training Planner

**Автор:** Колтман Владимир Николаевич

## Описание
Training Planner — это GUI-приложение на Python (Tkinter) для планирования тренировок. Программа позволяет добавлять тренировки с датой, типом и длительностью, фильтровать записи по типу и дате, а также сохранять и загружать данные через JSON.

## Возможности
- Добавление тренировки с полями: дата, тип, длительность.
- Проверка корректности ввода:
  - дата должна быть в формате `ГГГГ-ММ-ДД`;
  - длительность должна быть положительным числом.
- Фильтрация по типу тренировки и по дате.
- Сохранение и загрузка JSON.
- Удаление выбранной записи.

## Как использовать
1. Установите Python 3.
2. Запустите файл `training_planner.py`.
3. Введите дату, выберите тип тренировки и укажите длительность.
4. Нажмите **Добавить тренировку**.
5. Используйте фильтр по типу или дате.
6. Сохраняйте или загружайте данные через кнопки JSON.

## Примеры тестов
- Добавить запись: дата `2026-04-30`, тип `Кардио`, длительность `45`.
- Ввести дату `30-04-2026` — программа должна показать ошибку.
- Ввести длительность `-10` — программа должна показать ошибку.
- Отфильтровать тренировки по типу `Силовая`.
- Отфильтровать тренировки по дате `2026-04-30`.
